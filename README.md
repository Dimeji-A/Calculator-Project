# Python Calculator Project

## Description
Built a Python calculator from scratch with a GUI and keyboard input, handling addition, subtraction, multiplication, and division. Developed problem-solving, debugging, and event-driven programming skills.

## Features
- Custom GUI with clickable buttons and keyboard support
- Handles standard arithmetic operations (+, −, ×, ÷)
- Prevents invalid input and division by zero
- Modular code structure for readability and reusability
